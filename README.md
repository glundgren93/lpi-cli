# Usage
<!-- usage -->
```sh-session
$ npm install -g lpi
$ lpi COMMAND
running command...
$ lpi (-v|--version|version)
lpi/0.0.0 win32-x64 node-v12.18.3
$ lpi --help [COMMAND]
USAGE
  $ lpi COMMAND
...
```
