# Usage
<!-- usage -->
```sh-session
$ npm install -g lpi-helper
$ lpi-helper COMMAND
running command...
$ lpi-helper (-v|--version|version)
lpi-helper/0.0.0 win32-x64 node-v12.18.3
$ lpi-helper --help [COMMAND]
USAGE
  $ lpi-helper COMMAND
...
```
